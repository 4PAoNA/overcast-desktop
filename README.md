# overcast-desktop
Electron wrapper for Overcast.fm. There is no dist version of the app yet. To run it, follow the Development instructions below.

## Development
1. clone the repo
2. `npm install`
3. `npm start`

## Features
* hotkeys
* preferences including dark mode
